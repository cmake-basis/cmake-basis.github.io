Detailed Description

Basic command-line tools.

The BASIS package not only provides the implementation of the standardized build system, auxiliary implementations for different supported programming languages, and the standardized project directory structure, it also includes some utility command-line tools. In particular, the project management tool which is used to create and/or modify BASIS projects, and tools for the automated testing of the software of a BASIS project. These tools are summarized in this module.
